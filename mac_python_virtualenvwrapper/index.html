    설치¶

    pip 로 virtualenvwrapper 를 설치합니다.

    $ sudo easy_install pip
$ sudo pip install virtualenvwrapper

    bash 프로파일을 설정합니다.

    $ vim ~/.bash_profile
# python virtualenv
export WORKON_HOME=~/VirtualEnvs
source /usr/local/bin/virtualenvwrapper.sh

    터미널을 재시작합니다.
